# 🎓 SENAI-SP — Curso de IA com Google Antigravity

> Repositório de atividades práticas desenvolvidas durante o curso **Inteligência Artificial com Google Antigravity** no **SENAI-SP**.

---

## 📌 Sobre o Curso

O curso **IA com Google Antigravity** é uma iniciativa do SENAI-SP em parceria com o Google, com foco em capacitar alunos em desenvolvimento de software assistido por Inteligência Artificial, utilizando ferramentas modernas de IA generativa para criação de aplicações reais.

---

## 📁 Projetos
